# Personal Finance Dashboard

A Python-based personal finance analysis project that processes bank transaction CSV files, categorizes expenses, visualizes spending insights, and generates a PDF financial report.

# Features

-Automatic transaction categorization
-Spending analysis by category and month
-Data cleaning and preprocessing
-Visual charts using Matplotlib
-PDF report generation
-Modular structure for easy extension

# Technologies Used

-Python
-Pandas
-Matplotlib
-Jupyter Notebook
